# EMAIL

## Descripción

Este proyecto de JavaScript implementa un formulario de contacto interactivo. Permite al usuario ingresar su dirección de correo electrónico, asunto y mensaje, y luego enviar el formulario. El código valida la información ingresada por el usuario y muestra alertas de error si los campos están vacíos o si el formato del correo electrónico es incorrecto. Una vez que se completa el formulario y se envía con éxito, se muestra un mensaje de éxito al usuario.

## Instrucciones de uso

Para acceder a la página, simplemente abra el archivo index.html en su navegador web
